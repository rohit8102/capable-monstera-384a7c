# 🌐 Personal Resume Portfolio Website

## 📌 Project Overview
This project is a **personal portfolio website** developed as part of my BCA studies.  
It is designed to showcase my **resume, skills, and certificates** in a modern and interactive way.

The website follows a **dark modern developer theme** with smooth animations and a structured layout.

---

## 🚀 Features
- 🔐 Login Authentication (using Local Storage)
- 🏠 Home Page (Introduction / Portfolio)
- 📄 Resume Page (Academic & Personal Details)
- 📂 Certificates Section (13 PDF documents with live preview)
- 🎬 Smooth Animations & Transitions
- 📱 Responsive Design (Mobile Friendly)
- ⚡ Fast Navigation without page reload
